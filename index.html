Frontend Starter

前端学习，从会看页面开始

这是一个适合入门的静态学习页。你可以先理解页面结构，再观察样式怎么改变视觉效果，最后用一点 JavaScript 给页面加交互。前端的核心就是这三部分一起工作。

建议顺序：先学 HTML 语义标签，再学 CSS 盒模型与布局，最后学 JavaScript 如何选中元素并响应点击事件。
H

HTML 负责结构

HTML 决定页面里有什么内容，比如标题、段落、按钮、图片和列表。它像房子的骨架，让浏览器知道每一块内容是什么。

C

CSS 负责样式

CSS 决定页面看起来怎样，比如颜色、间距、排版、圆角和动画。没有 CSS，网页通常能用，但几乎没有视觉设计。

JS

JavaScript 负责交互

JavaScript 让页面动起来。点击按钮、展开说明、切换主题、提交表单验证，通常都需要 JavaScript 参与。

一个最小网页例子

左边是代码，右边是它渲染出来的大致效果。先看结构，再看样式，再想交互从哪里接进去。

<section class="demo-box">
  <h3>你好，前端</h3>
  <p>这是一个练习区块。</p>
  <button>点我试试</button>
</section>

<style>
  .demo-box {
    padding: 16px;
    border-radius: 16px;
    background: #fff4ea;
  }
</style>

<script>
  button.onclick = () => alert("开始学习");
</script>

你好，前端

这是一个练习区块。结构、样式、交互已经各自分工。

观察点：

标题来自 HTML，颜色与留白来自 CSS，点击行为来自 JavaScript。

入门学习清单

下面这几个主题适合作为第一周练习。完成后，你就能自己写简单静态页面了。

  • 认识常见 HTML 标签：`h1`、`p`、`a`、`img`、`button`、`div` 先学
  • 理解 CSS 选择器、盒模型、`margin`、`padding`、`display` 重点
  • 学习 JavaScript 基础：变量、函数、事件、DOM 选择 重点
  • 反复练习：模仿一个简单网页，把它拆成结构、样式、交互三层 持续做
